## Documentation des composants React du code du travail numérique

Composants [ReactJS](http://reactjs.org/) du site du [Code du travail numérique](https://codedutravail.num.social.gouv.fr).

Le CSS utilisé est celui de [code-du-travail-css](https://github.com/SocialGouv/code-du-travail-numerique/packages/code-du-travail-css), que vous devez importer séparément.

Contribuez au projet ici : [code-du-travail-ui](https://github.com/SocialGouv/code-du-travail-numerique/packages/code-du-travail-ui).

## Usage

```js
import { Container, Alert } "@cdt/ui"

const App = (
  <Container>
    <Alert primary>Primary alert message</Alert>
    <Alert secondary>Secondary alert message</Alert>
  </Container>
)
```

> Pour récupérer le CSS, voir le projet [code-du-travail-css](https://github.com/SocialGouv/code-du-travail-css).
